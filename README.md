# Fun Science PSLE — Videos, Study Guides & Games

Free PSLE Science revision for all levels, aligned to the MOE syllabus.
Each topic has a narrated video, a study guide, and a game (solo or duel).

## Live site
Once hosted with GitHub Pages: **https://funsciencepsle.github.io/**

| Topic | Page |
|---|---|
| Heat & Its Effects | /heatquest/ |
| The Water Cycle | /waterquest/ |
| Photosynthesis | /plantquest/ |
| Respiratory & Circulatory | /bodyquest/ |

Each topic folder contains: index.html (topic hub), game/index.html (the game),
video.mp4 + captions.srt (the video), and guide.html (the study guide).

## How to host (GitHub Pages)
1. Create a repository named exactly **funsciencepsle.github.io** (Public).
2. Upload the entire contents of this folder, keeping the folder structure intact.
3. Settings → Pages → Source: Deploy from a branch → main → / (root) → Save.
4. Wait 1–2 minutes, then visit https://funsciencepsle.github.io/

## Updating later
Replace any file (e.g. a game or video) and commit — the live site refreshes
within a minute or two.
